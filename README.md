# angular-initial

Simple AngularJS seed project to bootstrap your angular project using:
* Gulp
* Browser Sync
* Bower
* Less

```
  |  |  |  |- modules.js
  |  |  |  `- services.js
  |  |  |- module*/

  |- gulpfile.js
  `- package.json
```

## Usage
```
npm install
bower install

gulp help
```
